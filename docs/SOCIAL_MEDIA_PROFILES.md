## Consistent Branding Elements

### Brand Voice

| Attribute | Description |
|-----------|-------------|
| **Tone** | Warm, reassuring, simple |
| **Personality** | Trustworthy, focused, modern |
| **Language** | Plain English, no jargon |
| **Emotion** | Confidence, connection, peace of mind |

### Key Phrases (Use consistently)

| Phrase | Context |
|--------|---------|
| "See your child's day" | Tagline for parents |
| "Know their photos are safe" | Security messaging |
| "Free for families" | Pricing clarity |
| "No per-child fees" | Differentiator from competitors |
| "Privacy-first" | Brand positioning |
| "One thing perfectly" | Focus vs. all-in-one apps |

### Brand Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Violet (Primary) | #7C3AED | Headers, CTAs, accents |
| White | #FFFFFF | Backgrounds |
| Dark Gray | #1F2937 | Text |
| Light Gray | #F3F4F6 | Secondary backgrounds |

### Emoji Usage

Keep minimal and professional. Acceptable:
- Camera-related: 📷
- Security: 🔒
- Check marks: ✓ or ✔️
- Simple: 👶 (sparingly)

Avoid:
- Excessive emojis
- Trending/ironic emoji use
- Multiple emojis in succession
